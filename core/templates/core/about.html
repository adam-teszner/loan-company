About this website . . .

This website is for educational purposes only. The information provided on this site is not intended to be real. Any kind of similarities with real names, dates, numbers (such as id, pesel etc) should be treated as fictional.

The idea behind this website is solely for me to learn web development. I didn't want to create "another" online shopping or blog website the internet is full of tutorials of how to make one.

I am aware that some features used here are not the best practices or best use cases - it is meant for learning, understanding how things work - sometimes I wanted to solve a problem without the use of the internet but rather to think of solution myself. It might have produced unelegant, inefficient results like the table sorting in "My Customers"

Technologies used here are Python, Django for backend development - with some basic use of Django Rest Framework. Because I was starting from scratch and I had absolutely no clue about web developemnt I intentionally didn't use front-end css libraries like Bootstratp - just to learn some HTML and CSS basics. Some things in JS are written by myself instead of using prepared packages for example I wrote a script that generates HTML forms (with validation) from Json send by DRF - something similiar to JsonForms...
